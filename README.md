# GPUI 计算器应用

一个使用 GPUI 和 GPUI Component 构建的跨平台桌面计算器应用。

## 功能特性

### 1. 基本计算功能
- ✅ 加法、减法、乘法、除法运算
- ✅ 小数点支持
- ✅ 清除 (C) 和退格 (⌫) 功能
- ✅ 连续运算支持

### 2. 用户界面
- ✅ 双行显示屏：
  - 上方显示完整计算公式 (例如：1 + 2 / 3)
  - 下方显示当前数值
- ✅ 网格布局的数字和运算符按钮
- ✅ 不同颜色按钮分类：
  - 数字按钮：标准样式
  - 运算符按钮：主色调 (蓝色)
  - 清除按钮：危险色 (红色)
  - 退格按钮：警告色 (黄色)
  - 等号按钮：成功色 (绿色)

### 3. 主题支持
- ✅ 使用 Hybrid 主题
- ✅ 支持 Light 和 Dark 两种模式
- ✅ 主题文件位于 `themes/hybrid.json`
- ✅ 默认使用 Hybrid Dark 主题

### 4. 自适应设计
- ✅ 窗口自适应：使用 flexbox 布局
- ✅ 组件自动缩放以适应窗口大小
- ✅ 最大宽度限制确保在大窗口下保持合理尺寸

## 运行项目

```bash
# 构建项目
cargo build

# 运行应用
cargo run
```

## 技术栈

- **GPUI**: 高性能的 GPU 加速 UI 框架
- **GPUI Component**: UI 组件库
- **Rust**: 系统编程语言

## 项目结构

```
.
├── Cargo.toml          # 项目配置和依赖
├── src/
│   └── main.rs         # 主程序代码
└── themes/
    └── hybrid.json     # Hybrid 主题配置文件
```

## 使用说明

1. **数字输入**：点击数字按钮 (0-9)
2. **运算符**：点击 +、-、×、÷ 按钮
3. **小数点**：点击 . 按钮
4. **计算结果**：点击 = 按钮
5. **清除**：点击 C 按钮清除所有
6. **退格**：点击 ⌫ 按钮删除最后一位

## 示例计算

```
输入: 1 + 2 × 3
显示屏上方: 1 + 2 × 3 =
显示屏下方: 9
```
