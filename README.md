# GPUI Calculator

A cross-platform desktop calculator application built with GPUI and GPUI Component.

<img width="532" height="644" alt="Image" src="https://github.com/user-attachments/assets/baf65230-9d49-48bf-9347-5586af0bc0ad" />

## Running the Project

```bash
# Build the project
cargo build

# Run the application
cargo run
```

## Tech Stack

- **GPUI**: High-performance GPU-accelerated UI framework
- **GPUI Component**: UI component library
- **Rust**: Systems programming language
